# 智能排课系统毕业设计总结报告

## 一、项目概述

### 1.1 项目名称
**智能排课系统（Smart Schedule System V2.0）**

### 1.2 项目简介
本系统是一个基于 Spring Boot + React 的全栈智能排课管理系统，采用遗传算法实现自动排课功能，集成 Dify AI 工作流实现智能调课助手，为教务管理人员提供高效、智能的排课解决方案。

### 1.3 解决的问题
- **排课效率低**：传统手工排课耗时耗力，容易出错
- **资源冲突**：教室、教师、班级时间冲突难以避免
- **调课复杂**：临时调课需要大量人工协调
- **信息孤岛**：排课数据难以与其他系统联动
