## API 接口

> 接口详细信息请使用 MCP 工具从 OpenAPI Spec 获取

### 主要接口

| 功能 | 方法 | 路径 | 说明 |
|------|------|--------|------|
| 分页查询 | GET | `/v1/xxx/getPage` | 获取列表，支持筛选 |
| 添加 | POST | `/v1/xxx/add` | 添加新记录 |
| 更新 | PUT | `/v1/xxx/update` | 更新记录 |
| 删除 | DELETE | `/v1/xxx/delete` | 删除记录 |

### 分页查询接口

**路径**：`GET /v1/xxx/getPage`

**查询参数**：
- `page` (number, 必填) - 页码
- `size` (number, 必填) - 每页数量
- `xxx_uuid` (string, 可选) - UUID 筛选

**响应字段**：
- `total` - 总记录数
- `page` - 当前页码
- `size` - 每页记录数
- `records` - 数据列表
  - `uuid` - 记录UUID
  - `name` - 名称
  - ... 其他字段

### 添加接口

**路径**：`POST /v1/xxx/add`

**请求体字段**：
- `name` (string, 必填) - 名称
- ... 其他字段

### 删除接口

**路径**：`DELETE /v1/xxx/delete`

**查询参数**：
- `xxx_uuid` (string, 必填) - 记录UUID

## 命名规范

**重要**：
- **URL 查询参数**：蛇形命名（snake_case）
  - 示例：`course_type_uuid`, `relation_uuid`

- **请求体字段**：驼峰命名（camelCase）
  - 示例：`courseTypeUuid`, `classroomTypeUuid`
